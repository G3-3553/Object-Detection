HOW TO USE :
Upload an image using drag and drop option or using browse option.
Press 'DETECT OBJECTS' button to detect the objects in the image or press 'RESET' button to choose a new image.
The model will display a list of number of objects detected in the image. Press 'SHOW PREDICTIONS' button to visualize the output.
Press 'CHOOSE A NEW FILE' to detect objects in a new file.
